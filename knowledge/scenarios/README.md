# Scenarios

Use this directory for scenario-based guidance.

This is the best place for notes like:
- "I want to build a chatbot backend"
- "I need a lightweight internal tool"
- "I want to migrate from monolith to services"
- "I need to integrate an external API safely"

Each scenario note should include:
- problem framing
- recommended default approach
- alternative options
- prerequisites
- caveats
